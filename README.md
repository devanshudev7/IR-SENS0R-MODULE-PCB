# KiCad Project: Custom IR Sensor Module

Excited to share a new milestone in my hardware journey! 🚀 I recently started diving into PCB design using KiCad, and for my very first board, I designed a basic IR Sensor Module.

It has been incredibly rewarding to take this from a theoretical schematic down to routing the physical traces and finally seeing the 3D render come to life.

A few key takeaways from my first build:

✅ Understanding the core working principles of IR sensors and how to integrate them into a circuit.

✅ Schematic capture and footprint assignment.

✅ Component placement and trace routing.
